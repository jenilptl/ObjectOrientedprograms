# ObjectOrientedprograms-main 🚀

Welcome to **ObjectOrientedprograms**, a structured collection of Object-Oriented Programming concepts and practical implementations using Java.

## 📂 Repository Structure

The programs are organized into logical levels, progressing from basic syntax to advanced OOP concepts:

| Level | Focus Area | Key Concepts |
| :--- | :--- | :--- |
| **LEVEL 2** | Basics | Input/Output, Basic Calculations (Area, Temp) |
| **LEVEL 3** | Control Flow | Conditionals, Loops, Primes, Palindromes |
| **LEVEL 4** | Data Structures | Arrays, String Manipulation |
| **LEVEL 5** | Patterns | Diamond & Pyramid Shapes, Searching |
| **LEVEL 6** | Classes & Objects | Constructors, Statics, Complex Numbers, Bank Accounts |
| **LEVEL 7** | Inheritance | Base/Derived Classes, Coordinate Systems, Publication Tracking |
| **LEVEL 8** | Abstraction | Abstract Classes, Polymorphism (Vegetable Hierarchy) |

## 🛠️ Getting Started

### Prerequisites
- Java Development Kit (JDK) installed.
- Any text editor or IDE (VS Code, IntelliJ, Eclipse).

### Running a Program
1. Navigate to the desired level directory:
   ```bash
   cd "LEVEL 6"
   ```
2. Compile the Java file:
   ```bash
   javac ComplexTest.java
   ```
3. Run the compiled class:
   ```bash
   java ComplexTest
   ```
